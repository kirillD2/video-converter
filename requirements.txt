# Сюда записывают библиотеки, которые нужно скачать, чтобы всё заработало.

#
# Но для этого проекта ничего качать не нужно! :)
# Я использовал только стандартные модули, которые уже есть в Python:

#
# Как запустить:
# 1. Установите Python 3 (если его нет).
# 2. Установите FFmpeg:
#    - Windows: Скачайте с официального сайта (https://ffmpeg.org/download.html)
#      и добавьте путь к папке bin в переменные среды PATH.
#    - macOS (через Homebrew): brew install ffmpeg
#    - Linux (через apt): sudo apt update && sudo apt install ffmpeg
# 3. Запустите скрипт: python video_converter.py
